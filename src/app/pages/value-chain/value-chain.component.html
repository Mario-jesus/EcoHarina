Proceso

Etapas

  1. El proceso comienza con la recepción y limpieza de la pulpa de calabaza.
  2. La pulpa se muele y tamiza para reducir su tamaño y eliminar impurezas.
  3. Se seca para reducir la humedad y prevenir el crecimiento de microorganismos.
  4. La pulpa se muele finamente y se tamiza nuevamente para obtener la harina.

Tecnología

El proceso de producción de EcoHarina incluye clasificación óptica para eliminar impurezas, molienda por impacto para reducir el tamaño de partícula de manera eficiente, y secado con energía solar y secadores de lecho fluidizado para conservar energía.

Control de calidad

Durante la producción de EcoHarina, se inspecciona la pulpa de calabaza para verificar su calidad y se monitorean parámetros del proceso como temperatura, humedad y densidad. En el laboratorio, se analizan proteínas, grasas, fibra y cenizas para evaluar la calidad nutricional.
